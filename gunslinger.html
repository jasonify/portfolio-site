Brett Favre Gunslinger

A super fun project I did for Sqor with an amazingly talented artist, Nikita Orlov.

On this project I concentrated on coding the game in Swift (Apple's new language) and game design / game mechanics. With such a tiny team, we basically had to do a bit of everything and wear many hats.

The game was shown as an early preview at GDC 2015.
